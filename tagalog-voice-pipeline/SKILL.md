# Tagalog Voice Pipeline (Deployed: hulk-tagalog)

Status as of 2026-07-06: fully deployed and verified. This is the live system — use it directly,
don't rebuild it.

## The command

Everything routes through `~/hulk-scripts/hulk-tagalog`, backed by
`~/hulk-scripts/tagalog_engine.py`. Full docs at `~/hulk-scripts/TAGALOG_SYSTEM_README.md`.

| Subcommand | Function | Engine |
|---|---|---|
| `speak` | Text → spoken Tagalog | MMS-TTS (local, MPS) or gTTS (`--engine gtts`, cloud) |
| `listen` | Tagalog audio → text (optional `--translate` to English) | faster-whisper |
| `translate` | English ↔ Tagalog | Google Translate via deep-translator |
| `vocab` | Word/root/affix extraction | Custom analyzer |
| `vocab --flashcards` | Interactive HTML flashcards (HULKCLAW dark theme) | — |
| `content` | TikTok/IG/DJ drop script generation | Template + translate |
| `pronounce` | Reference audio → record → playback drill | gTTS + ffmpeg |
| `status` | Full health check | — |

### Examples

```bash
hulk-tagalog speak "Kumusta, mga kaibigan"
hulk-tagalog speak "Magandang gabi" --engine gtts
hulk-tagalog listen ~/Downloads/tagalog_podcast.mp3
hulk-tagalog listen ~/Downloads/interview.wav --translate
hulk-tagalog translate "I love making beats" --to tl
hulk-tagalog translate "Ang ganda ng araw ngayon" --to en
hulk-tagalog vocab --from ~/studio/voice-training/tagalog-mine/sentences.txt --flashcards
hulk-tagalog content --type tiktok --topic "beat flipping"
hulk-tagalog content --type djdrop --topic "new sample pack"
hulk-tagalog pronounce --text "Maraming salamat sa pagbili"
```

## Hard rule — commercial licensing (unchanged, still enforce)

- **NEVER** ship, sell, or distribute audio generated by the `speak` subcommand's default
  MMS-TTS engine. It is CC-BY-NC — social/personal use only.
- Commercial-safe options right now: `gTTS` (`--engine gtts`, cloud, free tier) or the custom
  trained voice from `tagalog-mine/` once recorded.
- Before generating TTS for anything buyer-facing (product, funnel, sample pack tag, DJ drop
  meant for sale) — confirm which engine was used. If MMS-TTS, block it and route to gTTS or
  flag that the custom voice isn't ready yet.

## The one open task

`tagalog-mine/` (record.py + sentences.txt) is the road to a 100%-sellable Tagalog voice —
Piper fine-tune from an `id_ID` checkpoint. Nothing else is blocking it except actually recording
~30 minutes of audio against the 98 prepared sentences. When the user is ready, walk them through
`record.py`.

## Content workflow (already wired)

1. `hulk-tagalog content --type tiktok --topic "..."` — generate script
2. `hulk-tagalog speak "[script]" --engine gtts` — voice it (commercial-safe)
3. Import into Logic Pro, layer with beat
4. Post to TikTok/Reels/YouTube

## What to do when this skill activates

1. Default to `hulk-tagalog` subcommands — don't propose rebuilding pieces of this.
2. If output is commercial/buyer-facing, force `--engine gtts` (or ask if the custom voice is
   trained yet) — never let MMS-TTS output ship silently.
3. If the user wants a fully commercial local voice, the path is: finish recording in
   `tagalog-mine/`, then fine-tune Piper from `id_ID`. Don't suggest `tl_PH-nanay-medium` — it
   doesn't exist.
4. For anything not covered by the 7 subcommands, check `TAGALOG_SYSTEM_README.md` before
   improvising a new script.
