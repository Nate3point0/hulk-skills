# Metricool Social Strategist

**Trigger:** When scheduling social media content, analyzing engagement metrics, or planning multi-platform campaigns

**Platforms:** TikTok, Instagram, Threads, YouTube (via Metricool API)

**MCP Required:** Yes — Metricool MCP server with API credentials

## What It Does

Manages social media scheduling and strategy across TikTok, Instagram, Threads, and YouTube. Analyzes engagement patterns, suggests optimal posting times, plans content calendars, and tracks performance across platforms. Pulls metrics and sends recommendations via Claude.

## How to Use

1. Upload content (image/video + caption) or describe the post
2. Claude checks your follower demographics and historical engagement patterns
3. Claude suggests best posting time, hashtag strategy, and cross-platform tweaks
4. Schedule via Metricool — Claude handles platform-specific formatting
5. Track performance and get daily/weekly insights

## Notes

- Metricool API rate limits: 1000 requests/day on business plans
- TikTok algorithm: first 3 hours are critical; post when followers are most active
- Instagram Reels: vertical video, 15-60 sec, trending audio drives reach
- Threads: short-form, reply-heavy; engagement matters more than follower count
- Best times: Tue-Thu 6-9pm EST for max reach across all platforms
- Hashtag strategy: 5-10 relevant hashtags per post, mix trending + niche
- Content calendar: plan 2 weeks ahead to catch trends early
- Scheduling buffer: post 15 min before peak time to catch algorithm boost
